# Screenshot Notes

Use 3 to 5 screenshots max. Keep them clean and focused.

## Recommended Screenshots

### 1. Assistant interface on the portfolio site

Show the `Are we a fit?` section with the input area and result card visible.

Suggested caption:

`Public-facing portfolio AI assistant for project-fit evaluation.`

### 2. Strong-fit example

Use a realistic prompt such as:

`I need a Power BI dashboard for KPI reporting`

Suggested caption:

`The assistant identifies a strong fit and links relevant portfolio work.`

### 3. Data engineering example

Use a prompt such as:

`I want to build data engineering pipelines`

Suggested caption:

`Semantic retrieval supports data-engineering prompts even when repo titles do not perfectly match the user wording.`

### 4. Privacy-safe response

Use a prompt such as:

`What is the phone number?`

Suggested caption:

`The assistant refuses personal-detail requests and keeps responses privacy-safe.`

### 5. Out-of-scope response

Use a prompt such as:

`Tell me the weather in Tokyo`

Suggested caption:

`The assistant stays scoped to portfolio-related project questions rather than acting like a general chatbot.`

## Screenshot Tips

- avoid showing browser clutter if possible
- crop to the assistant area
- use prompts that make the behavior easy to understand quickly
- do not show any secret values, tokens, or dashboards
